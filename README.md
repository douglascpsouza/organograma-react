# Organo - React

A simple org chart maker.

## Description

Organo assists in the organization of teams, squads ou user groups.

## Getting Started

### Dependencies

* [Node](https://nodejs.org/)
* [React](https://react.dev/)

### Installing

* Clone this repo
* Inside the project's folder, run:
```
npm install
```

### Executing program

* To run the program, execute:
```
npm start
```
